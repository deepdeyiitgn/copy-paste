# QlynkCopy - Universal Clipboard Downloader 📋✨

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Author](https://img.shields.io/badge/author-Deep%20Dey-indigo.svg)

QlynkCopy is a lightning-fast, 100% client-side web application that allows users to instantly paste and download their clipboard contents (images, text, files) directly from their browser. Built with privacy and speed in mind, nothing is ever uploaded to a server.

🌍 **Live Demo:** [copy.qlynk.me](https://copy.qlynk.me/)

## 🚀 Features
* **100% Privacy:** All processing happens in the browser. No server uploads.
* **Universal Support:** Extracts images (PNG/JPG/WebP/GIF), text, and files.
* **Mobile Optimized:** Dedicated "Open Keyboard" feature allows mobile users to easily utilize their clipboard history.
* **Drag & Drop:** Supports dropping files directly into the manual paste zone.
* **SEO Optimized:** Fully marked up with Canonical links, OpenGraph tags, Twitter Cards, and Schema.org JSON-LD.
* **No Dependencies:** Runs entirely on a single `index.html` file using Tailwind CSS via CDN.

## 🛠️ How to Use
Since this project is a single-file application, installation is instantaneous:
1. Clone the repository or download `index.html`.
2. Open `index.html` in any modern web browser.
3. Start pasting!
